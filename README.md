# Calculadora Java Swing

Desktop calculator developed in Java with Swing graphical interface, built in 2022 as part of my Java learning path.

## Features
- Basic operations: addition, subtraction, multiplication and division
- Decimal number support
- Clear button to reset the display

## Technologies
- Java
- Java Swing
- Object-Oriented Programming

## Project Structure
- `Pantalla` - Calculator display
- `Botones` - Numeric keypad
- `OperacionesBotones` - Math operators
- `Interfaz` - Main window and calculation logic
